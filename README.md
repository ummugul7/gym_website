Spor Salonu Yönetim Sistemi
spor salonu üyelerinin  randevu alabildiği, admin eğitmen, randevu ve üyeleri yönetebildiği web uygulaması. ASP.NET Core ile MVC mimarisi kullanılarak geliştirilmiştir.

Özellikler
Üye (Kullanıcı) Paneli

Giriş & Kayıt: ASP.NET Core Identity altyapısı ile güvenli üyelik sistemi.

Randevu Sistemi: Mevcut dersler veya eğitmenler için randevu oluşturabilme.

Kişisel Takip: Alınan randevuların listelenmesi ve yönetimi.

Admin Paneli

Eğitmen Yönetimi: Yeni eğitmenlerin sisteme eklenmesi, düzenlenmesi ve silinmesi.

Randevu Planlama: Randevu oluşturma.

Sistem Takibi: Sistemdeki genel durumun üye sayısı, aktif randevular vb. takibi.

 Kullanılan Teknolojiler
Framework: .NET Core (Razor Pages & MVC)

Kimlik Doğrulama: ASP.NET Core Identity (Authorization & Authentication)

Frontend: HTML5, CSS3, Bootstrap

Veritabanı: Entity Framework Core 


Uygulamayı çalıştırın: dotnet run
